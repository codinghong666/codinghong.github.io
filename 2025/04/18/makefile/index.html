3 预定义变量

  • $@: 目标(target)的完整名称
  • $<: 第一个依赖文件（prerequisties）的名称
  • $^: 所有的依赖文件（prerequisties），以空格分开，不包含重复的依赖文件

 

2.Makefile 的常用函数

函数调用，很像变量的使用，也是以 “$” 来标识的，其语法如下：

  • fn: 函数名
  • arguments: 函数参数，参数间以逗号 , 分隔，而函数名和参数之间以“空格”分隔

 

1 shell

  • 名称：shell 命令函数 —— shell
  • 功能：调用 shell 命令 command
  • 返回：函数返回 shell 命令 command 的执行结果
